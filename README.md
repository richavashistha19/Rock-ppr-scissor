# Rock-ppr-scissor

This is a simple implementation of the classic Rock-Paper-Scissors game using JavaScript. It's designed to help beginners learn basic logic building in JavaScript.

## How to Play

To play the game, simply open the `index.html` file in your web browser. The game will start immediately.

- You'll be prompted to choose between Rock, Paper, or Scissors.
- The computer will randomly choose its move.
- The result (win, lose, or draw) will be displayed on the screen.

## Features

- Simple interface for easy gameplay.
- Random computer moves for unpredictability.
- Outcome detection based on basic game rules.
- Score tracking to keep track of wins, losses, and draws.
- Reset button to start a new game.

## Technologies Used

- HTML: for the structure of the game.
- CSS: for styling the game interface.
- JavaScript: for game logic and interactivity.
